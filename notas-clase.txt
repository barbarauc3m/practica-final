OPERACIONES A HACER:

Gestión de usuarios:
    REGISTER: Registrar nuevo usuario

    UNREGISTER: Eliminar usuario

    CONNECT: Conectarse al sistema (indica IP/puerto)

    DISCONNECT: Desconectarse

Gestión de archivos:
    PUBLISH: Publicar nombre de archivo (no el contenido)

    DELETE: Eliminar archivo publicado

    LIST_USERS: Listar usuarios conectados

    LIST_CONTENT: Listar archivos de un usuario

    GET_FILE: Descargar archivo de otro cliente (transferencia directa P2P)
